What Is Class.?
<br>
Class is a Blue Print/ Master Plan which used to create the object,and it is used to create the "N" number of objects.
<br>
<b> Points to know</b>
<br>
-->Without Class you cannot create object
<br>
-->Class in python always starts with a Keyword "class" followed by class name and colon inside which we can write a logic in an indented block
<br>
-->This block is known as class scope or class block
<br>
<b>Syntax of class</b>
<br>
